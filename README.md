
# Email Generator Application

## Description

Email Generator Application is a Core Java-based console application developed to automate organizational email account creation for employees. The application generates official email addresses based on employee details and department selection.

The system also provides functionalities such as random password generation, verification code generation, alternate email configuration, password management, and employee information display through a menu-driven interface.



## Features

* Official Email Generation
* Department-Based Email Creation
* Random Password Generation
* Verification Code Generation
* Password Management
* Alternate Email Configuration
* Employee Information Display
* Menu-Driven Console Application

## Functionalities

### Email Generation

Automatically generates official email addresses using:
firstname.department@companydomain

### Password Generation

Generates random secure passwords using uppercase letters, numbers, and special characters.

### Verification Code

Creates verification codes for password-related operations.
